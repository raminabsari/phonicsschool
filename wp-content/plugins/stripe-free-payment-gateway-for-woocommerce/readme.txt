== Description ==

This plugin extends WooCommerce allowing user to take payments directly on your store via Stripe API. Stripe payment is available in the United States, Canada, UK, Australia, France, Belgium, Netherlands, Germany, Irelandand more

== Installation ==

1) Upload 'woocommerce-stripe' folder to the '/wp-content/plugins/' directory
2) Activate 'woocommerce-stripe' form wp plugin option in admin area
3) Plugin will appear in the menu bar of WP Dashboard
4) set the plugin settings from WooCommerce checkout setting
